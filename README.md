## Getting Started

Clona el repositorio.

```bash
git clone https://github.com/akaXala/proyectoAyDS.git
```

Para el instalar y ejecutar el proyecto usamos **yarn**.

```bash
npm install # Descarga las dependencias necesarias del proyecto
```
```bash
npx expo start 
```
